# Blender Note

## Shortcut

> 小键盘 1-9 可以进行正、俯、侧等角度切换。

> 没有小键盘，就在 edit > preferences > input > keyboard > check [Emulate Numpad] 或者点击右上角 ±x、y、z。

> Crtl + Alt + Q 三视图 + 透视图旋转。

> N 弹 sidebar。

> Alt + G 重置 grab（抓取）
> Alt + R 重置 rotate
> Alt + S 重置 scale
> 注意：要先选中要重置的对象。

> 查看某个对象的便捷，右侧 Object Properties 中，选择 Viewport Display，勾选 Bounds（边界框）。
